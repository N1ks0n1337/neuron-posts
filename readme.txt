A Neuron Expert WordPress plugin to display user posts.

== Description ==

This plugin allows you to display user posts on your WordPress website using the Neuron Expert API. Users can customize the display by providing their API key.

**This plugin relies on the Neuron Expert API service.**

For more information about the Neuron Expert service, please visit:
- [Neuron Expert](https://neuron.expert/)
- [Privacy Policy](https://neuron.expert/privacy/)
- [Terms and Conditions](https://neuron.expert/tc/)
== Installation ==

1. Upload the `neuron-posts` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

To display user posts, use the following shortcode:

[display_user_posts api_key="YOUR_API_KEY"]

Replace `YOUR_API_KEY` with the actual API key of the user.
